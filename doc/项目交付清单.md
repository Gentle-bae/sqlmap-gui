## 🔧 环境要求

### 最低配置

- **操作系统**: Windows 7+ / Linux / macOS
- **Python**: 3.6 及以上
- **内存**: 512MB 最小（建议1GB+）
- **磁盘空间**: 100MB

## 🚀 安装验证步骤

### 快速验证程序正常运行

```bash
# 步骤1：检查Python版本
python --version
# 输出：Python 3.6+ 以上

# 步骤2：安装依赖
pip install requests flask

# 步骤3：启动GUI
python sqlmap-gui.py
# 输出：GUI窗口打开

# 步骤4：启动靶场
python test_sqli_server.py
# 输出：Running on http://127.0.0.1:5000

# 步骤5：测试扫描
# 在GUI中输入：http://127.0.0.1:5000/Less-1/?id=1
# 点击开始扫描，应该发现SQL注入漏洞
```
